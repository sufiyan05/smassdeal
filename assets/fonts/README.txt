Add your custom font files here using these names:

- body-font.woff2
- body-font.woff
- heading-font.woff2
- heading-font.woff

How it works:

- `body-font` is used for the general UI text.
- `heading-font` is used for headings and key labels.
- You can add only `.woff2`, only `.woff`, or both.
- If these files are not present, the panel falls back to local Inter or the system sans-serif stack automatically.

Recommended:

- Use `.woff2` whenever possible.
- Keep the font names exactly as listed above.
